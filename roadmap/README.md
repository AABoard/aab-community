# Public Collaboration Roadmap

This roadmap describes collaboration priorities. It is not a promise of delivery dates.

## Near-Term Priorities

- Route contributors into clear working groups.
- Improve case and pilot contribution guidance.
- Recruit standards reviewers with K-12, learning science, assessment, and AI education expertise.
- Improve data schema and public export documentation.
- Develop policy monitoring and research synthesis workflows.

## Medium-Term Priorities

- Public comment process for standards drafts.
- Reviewer onboarding packets.
- Regular dated public registry exports.
- Public issue labels and triage workflow.
- Contributor-friendly examples for cases, frameworks, policies, and resources.

## How To Suggest Roadmap Changes

Open an issue describing the proposed priority, affected working group, public value, and any risks or dependencies.
